# She Can Foundation Website

A modern and responsive NGO website created for the **Web Development Internship Task**.

The project is based on the theme of women empowerment and community support through the fictional **She Can Foundation**.

---

##  Internship Task

Create a simple website for **She Can Foundation** using your own creativity.

### Required Sections
- Home Section
- About Section
- One Image
- Volunteer / Join Us Button
- Footer

### Additional Features Added
- Responsive Design
- Smooth Animations
- Interactive UI
- Custom Cursor Effect
- Scroll Reveal Animations
- Animated Statistics Counter
- Modern Glassmorphism Navbar
- Gradient Background Blobs

---

##  About the Task

As an NGO, the purpose of this task is to encourage students to learn and participate without fear of rejection or unrealistic expectations.

This project focuses on:
- Creativity
- Clean UI Design
- Responsiveness
- Frontend Development Skills
- Interactive User Experience
